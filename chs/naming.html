# 剧集命名规则

🟢 刮削剧集，文件名至少包含x集/話/话,EPxx
🟢 如文件名只有x集/話/话 EPxx，不含x季,SxxExx，则默认识别为第1季的第x集
🟢 精确到刮削x季x集，文件名必须包含SxxExx x季x集 x季x集/话/話 x季x

# 自定义水印图片文件

  • 建议大小为769 x 374或比例相近的PNG图片（必须包含透明图层否则读取失败）
