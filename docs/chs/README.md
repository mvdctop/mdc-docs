## 项目简介

Movie Data Crawler (MDC) 是一款用于整理与刮削**本地影片或剧集**文件的软件工具，可从互联网获取**电影或剧集**元数据并下载封面。

MDC 由两个部分组成：

* 网页端用户面板
* 本地客户端

### 用户面板

* **账户管理**：提供用户注册和登录功能
* **云配置管理**：管理云刮削相关配置
* **设备管理**：查看与管理已登录账号的设备

### 本地客户端

* ✅ **OpenAI 大模型翻译接入**：支持使用 OpenAI 大模型进行翻译
* ✅ **多平台支持**：支持 Windows、Linux、Docker、MacOS X86 & ARM
* ✅ **云配置同步**：同步网页端云配置，并根据配置整理本地影片
* ✅ **元数据获取与封面下载**：根据云配置刮削本地影片元数据并下载封面
* ✅ **多客户端支持**：支持同一账户在多个本地客户端上使用多个云配置实例

## 1. 下载并运行

* 前往[下载页面](https://dl.mvdc.top)下载并运行程序，建议全局代理使用
* 首次打开时，可在`Language`栏中选择语言：`简体中文` `繁體中文` `English`
* 除 Windows 系统外，下载前请阅读[各客户端使用方式](/chs/clients.html)
* 强烈建议安装`Chrome`浏览器，MDC 依赖`Chrome`提升刮削成功率

## 2. 注册账号并登录

* 在网页端[用户面板](https://user.mvdc.top)注册账号
* 在客户端中点击`登录`

![](/images/readme1.png)

登录后可进行个性化[云配置](https://user.mvdc.top/configuration/general)，新用户建议先使用当前默认配置。

* 云配置支持多个配置实例，可按不同影片类型添加多个实例
* 每个账号最多可登录 3 台设备，两种配置各最多 6 个云配置实例
* （可选）如果频繁使用或对数据精度有要求，建议申请 [TMDB API KEY](/chs/configuration.html#tmdb-api-key) 并填入云配置

## 3. 激活订阅

* 在[激活订阅](https://user.mvdc.top/activation)页面选择适合的订阅计划
* 付款完成后，订阅会自动激活
* 支持多种支付方式

## 4. 配置本地目录

* 在`本地配置`页面中配置刮削目录与输出目录
* NAS 或云盘用户请参考[NAS 或云盘的目录配置](/chs/cloud_mount.html)

![](/images/readme2.png)

默认情况下，输入文件夹为程序所在目录，输出文件夹为程序所在目录下新建的`output`文件夹。

## 5. 选择刮削类型

根据[文件命名规则](/chs/naming.html)，在`本地配置`页右下角点击相应`刮削类型`。

![](/images/scraping_type.png)

* 建议处理影片前点击`名称提取检查`，确认提取结果是否符合预期
* 如果发现提取名称不符合预期，请手动重命名影片文件

### 外挂字幕文件

若`刮削目录`中的字幕文件名前缀与影片文件名前缀一致，后续处理时会自动将字幕移动到影片同一目录，并在 NFO 元数据文件中添加`字幕`标签，同时给封面打上水印（可在网页端云配置页面中打开或关闭）。

## 6. 开始刮削

如果已在客户端**登录后**修改了网页端云配置，请先在**客户端**`配置`页点击`同步`，再点击`开始`运行。

::: tip
* 支持将多个或单个文件拖动至 MDC 窗口中刮削
* 单个文件拖拽到 MDC 窗口时，可自定义名称刮削
:::

处理失败的影片会被记录，并在程序所在目录下生成`failed_list.txt`失败列表文件，可在本地配置页中移除。

## 7. 整理完成

::: warning
* 请仅将整理前后的文件与产生的元数据用于个人本地管理  
* 请勿传播或分发，这可能会在用户所在地产生法律责任
:::

可将输出文件夹导入 Emby、Jellyfin、Plex 等媒体管理器中管理影片。
