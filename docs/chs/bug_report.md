# 🐞 报告 BUG

## 邮件标题

```text
[BUG报告]XXX
```

::: warning
未订阅，或无 2 张及以上截图用于判断 BUG，可能无法回复。
:::

## 正文

* 显示阁下**已订阅**的截图，或直接从订阅邮件下方回复

### BUG 描述

* **只发送一句话且没有截图，无法准确判断 BUG 构成原因**
* 清晰、简洁地描述错误是什么
* 发送 2 张及以上**截图**，如云配置、本地配置、运行输出、文件处理前后截图
