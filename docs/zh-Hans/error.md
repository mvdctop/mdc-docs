# 🚨 异常处理

## 打开客户端无任何反应

若运行 `MDC.exe` 或 `MDC.app` 后没有任何反应，请先按系统类型[清空配置](/zh-Hans/clean_config.html)，然后重新启动 MDC 客户端

## 打开客户端后，窗口内容无法显示

请按以下步骤排查

1. 确认客户端压缩包已完整解压，不要直接在压缩包内点击 exe 文件运行
2. 临时关闭杀毒软件或安全防护软件后，重新打开客户端

## 连接超时

如果日志中出现以下重试信息

```text
[-]Connect retry 1/3
[-]Connect retry 2/3
[-]Connect retry 3/3
```

或出现以下错误

```text
Caused by ProxyError
```

请检查代理设置，并按以下方式处理

1. 重新打开代理软件
2. 逐个更换代理节点后重试
3. 如果未开启自定义代理，请开启全局代理后再运行客户端
