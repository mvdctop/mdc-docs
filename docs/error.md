# 🚨 Exception Handling

## Opening the Client Shows No Response

If running `MDC.exe` or `MDC.app` shows no response, first [clear the configuration](/clean_config.html) according to your system type, then restart the client.

## `Caused by ProxyError`

Please check the proxy. If the custom proxy is not enabled, enable the global proxy and run again.

If the error persists, switch to another proxy node.
