# 🔔 處理結果推送

客戶端每次處理文件完成後，可推送處理結果，包括成功和失敗的文件數量。

## 接入教學

### 取得 Chat ID

登入 Telegram，進入[機器人](https://t.me/mvdcbot)，點擊底部 `START` 按鈕，或輸入`/start`，即可取得 9 位 Chat ID。

### 輸入 Chat ID

在[用戶面板 - 處理結果推播](https://user.mvdc.top/bot)頁面中，輸入取得的 Chat ID，勾選啟用，並選擇適合閣下的推播語言。

### 更新配置

如果閣下收到機器人發送的`Hello`訊息，即表示設定成功。

### 推送檔案名稱列表

每次推送結果都會包含成功或失敗的檔案名稱列表。

::: tip
我們僅進行資訊中轉推送，不會收集推送內容。
:::
