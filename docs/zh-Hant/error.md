# 🚨 錯誤處理

## 開啟客戶端無任何反應

若執行 `MDC.exe` 或 `MDC.app` 後沒有任何反應，請先按系統類型[清空配置](/zh-Hant/clean_config.html)，然後重新啟動 MDC 客戶端

## 開啟客戶端後，視窗內容無法顯示

請按以下步驟排查

1. 確認客戶端壓縮包已完整解壓，不要直接在壓縮包內點擊 exe 檔案執行
2. 暫時關閉防毒軟體或安全防護軟體後，重新開啟客戶端

## 連線逾時

如果日誌中出現以下重試資訊

```text
[-]Connect retry 1/3
[-]Connect retry 2/3
[-]Connect retry 3/3
```

或出現以下錯誤

```text
Caused by ProxyError
```

請檢查代理設定，並按以下方式處理

1. 重新開啟代理軟體
2. 逐個更換代理節點後重試
3. 如果未開啟自訂代理，請開啟全域代理後再執行客戶端
