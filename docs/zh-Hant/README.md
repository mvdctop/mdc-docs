## 專案簡介

Movie Data Crawler (MDC) 是一款用於整理與刮削**本地影片或劇集**文件的軟體工具，可從互聯網獲取**電影或劇集**元數據並下載封面。

MDC 由兩個部分組成：

* 網頁端用戶面板
* 本地客戶端

### 用戶面板

* **帳戶管理**：提供使用者註冊和登入功能
* **雲端配置管理**：管理雲端刮削相關配置
* **設備管理**：查看與管理已登入帳號的設備

### 本地客戶端

* ✅ **OpenAI 大模型翻譯接入**：支援使用 OpenAI 大模型進行翻譯
* ✅ **多平台支援**：支援 Windows、Linux、Docker、MacOS X86 & ARM
* ✅ **雲端配置同步**：同步網頁端雲配置，並根據配置整理本地影片
* ✅ **元資料取得與封面下載**：根據雲端配置刮削本地影片元資料並下載封面
* ✅ **多客戶端支援**：支援同一帳戶在多個本地客戶端上使用多個雲端配置實例

## 1. 下載並執行

* 前往[下載頁面](https://dl.mvdc.top)下載並執行程式，建議全域代理使用
* 首次開啟時，可在`Language`欄中選擇語言：`简体中文` `繁體中文` `English`
* 除 Windows 系統外，下載前請閱讀[各客戶端使用方式](/zh-Hant/clients.html)
* 強烈建議安裝`Chrome`瀏覽器，MDC 依賴`Chrome`提升刮削成功率

## 2. 註冊帳號並登入

* 在網頁端[用戶面板](https://user.mvdc.top)註冊帳號
* 在客戶端中點擊`登入`

![](/images/readme1.png)

登入後可進行個性化[雲配置](https://user.mvdc.top/configuration/general)，新用戶建議先使用當前預設配置。

* 雲端配置支援多個配置實例，可按不同影片類型新增多個實例
* 每個帳號最多可登入 3 台設備，兩種配置各最多 6 個雲端配置實例
* （可選）如果頻繁使用或對資料精度有要求，建議申請 [TMDB API KEY](/zh-Hant/configuration.html#tmdb-api-key) 並填入雲端配置

## 3. 激活訂閱

* 在[激活訂閱](https://user.mvdc.top/activation)頁面選擇適合的訂閱方案
* 付款完成後，訂閱會自動激活
* 支援多種支付方式

## 4. 配置本地目錄

* 在`本地配置`頁面中配置刮削目錄與輸出目錄
* NAS 或雲端硬碟用戶請參考[NAS 或雲端硬碟的目錄配置](/zh-Hant/cloud_mount.html)

![](/images/readme2.png)

預設情況下，輸入資料夾為程式所在目錄，輸出資料夾為程式所在目錄下新建的`output`資料夾。

## 5. 選擇刮削類型

根據[檔案命名規則](/zh-Hant/naming.html)，在`本地配置`頁右下角點擊相應`刮削類型`。

![](/images/scraping_type_zh-Hant.png)

* 建議處理影片前點擊`名稱提取檢查`，確認提取結果是否符合預期
* 如果發現提取名稱不符合預期，請手動重命名影片檔案

### 外掛字幕文件

若`刮削目錄`中的字幕檔名前綴與影片檔名前綴一致，後續處理時會自動將字幕移動到影片同一目錄，並在 NFO 元資料檔中添加`字幕`標籤，同時給封面打上浮水印（可在網頁端雲端配置頁面中開啟或關閉）。

## 6. 開始刮削

如果已在客戶端**登入後**修改了網頁端雲端配置，請先在**客戶端**`配置`頁點擊`同步`，再點擊`開始`執行。

::: tip
* 支援將多個或單一檔案拖曳至 MDC 視窗中刮削
* 單一檔案拖曳到 MDC 視窗時，可自訂名稱刮削
:::

處理失敗的影片會被記錄，並在程式所在目錄下產生`failed_list.txt`失敗列表檔案，可在本地配置頁中移除。

## 7. 整理完成

::: warning
* 請僅將整理前後的文件與產生的元數據用於個人本地管理
* 請勿傳播或分發，這可能會在使用者所在地產生法律責任
:::

可將輸出資料夾匯入 Emby、Jellyfin、Plex 等媒體管理器中管理影片。
