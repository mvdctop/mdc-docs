# 🗄️ NAS 的使用

## Windows

* 對應網路磁碟機：開啟檔案總管，在視窗頂部點擊`本機`，在下方點擊`對應網路磁碟機`

![](/images/network_drive.png)

* 請確保在`本機`中，**NAS 磁碟位於`網路位置`**，而不是`裝置與磁碟機`
* 程式放置在目前電腦本地磁碟中；若 NAS 共享已開啟 exe 執行權限，也可放置在網路磁碟機中
* 執行客戶端`MDC.exe`，刮削目錄和輸出目錄選擇網路磁碟機下的目錄

## macOS

* 參考[將你的 Mac 連接到共享電腦和伺服器](https://support.apple.com/zh-tw/guide/mac-help/mchlp1140/mac)
* 執行客戶端`MDC`，刮削目錄和輸出目錄選擇已連接伺服器中的資料夾

# 🗂️ 雲端碟掛載工具使用

由於雲端碟廠商流控，請避免在雲端碟中建立元資料相關文件。強烈建議閣下調整為以下配置。

::: tip
Windows 請確保在`本機`中，NAS 磁碟位於`網路位置`，而不是`裝置與磁碟機`。
:::

如果閣下使用 [CloudDrive](https://www.clouddrive2.com/)、`RaiDrive`、`Alist` 等本地雲端碟掛載工具，請確保在掛載工具中，掛載的碟符類型為`網路碟`，否則可能會導致程式執行出錯。

程式請放置在本地磁碟中啟動。

![](/images/mount.jpg)

## 雲端配置調整建議

* 常規設定 - 整理模式 - 選擇`創建strm檔案`

## 本機配置調整建議

* 目錄配置 - 刮削目錄 - 選擇**雲端碟**中的目錄
* 目錄配置 - 輸出目錄 - 選擇**本地**的目錄
