# 🐞 報告 BUG

## 郵件標題

```text
[BUG報告]XXX
```

::: warning
未訂閱，或無 2 張及以上截圖用於判斷 BUG，可能無法回覆。
:::

## 正文

* 顯示閣下**已訂閱**的截圖，或直接從訂閱郵件下方回覆

### BUG 描述

* **只發送一句話且沒有截圖，無法準確判斷 BUG 構成原因**
* 清晰、簡潔地描述錯誤是什麼
* 發送 2 張及以上**截圖**，如雲端配置、本地配置、運行輸出、文件處理前後截圖
